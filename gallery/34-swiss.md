# DESIGN.md — スイス・インターナショナル

---

## Project Overview

- **Project Name**: スイス・インターナショナル — グリッドとヘルベチカ
- **Vibe**: "厳密なグリッド、ヘルベチカ、赤×白×黒。情報伝達の機能主義を極めたタイポグラフィデザイン"
- **Target User**: 美術館、建築事務所、タイポグラフィ重視のサイト

---

## Color Palette & Roles

```
Primary:      #FF0000   （アクセント・スイスレッド）
Secondary:    #000000   （テキスト・ボーダー）
Background:   #FFFFFF   （ページ背景）
Surface:      #F0F0F0   （セクション背景）

```


---

## Do's and Don'ts

### Do
- グリッドを厳密に守る
- テキストは左揃えに統一
- 赤は最小限のアクセントとして使う

### Don't
- 中央揃えを多用しない
- 装飾的なフォントを使わない
- 角丸を使わない
