## DESIGN.md の可能性

今日のセミナーを通じて、**DESIGN.md というたった1つのMarkdownファイル**が、AIエージェント時代のデザインワークフローを根本から変える可能性を実感しました。

### 気づき

これまでAIにデザインを頼むたびに、色もフォントもレイアウトもバラバラなものが出てきていました。原因はシンプルで、**AIにはデザインの「正解」を渡していなかった**ということです。

DESIGN.md は、その正解を **人間が定義し、AIが忠実に再現する** という新しい分業を実現するツールでした。

### 驚いたこと

- **Markdownだけで完結する**: Figmaもデザインツールも不要。テキストエディタで書ける
- **AIエージェントが自動で読む**: プロジェクトルートに置くだけで、Claude Code が即座に参照する
- **一貫性が保たれる**: 何度AIに頼んでも、同じカラー・フォント・余白でコードが生成される
- **デザイナーがいなくても使える**: テンプレートを選ぶだけで、プロ品質のデザインシステムが手に入る

### 特に印象的だった概念

> **「人間が判断し、AIが再現する」**

デザインの判断（色・フォント・余白のルール）は人間が行い、その実装（HTML/CSS生成）はAIが行う。この分業こそが、AIエージェント時代のデザインの本質だと感じました。

## 実践: 制作したサイト

セミナーの内容を元に、DESIGN.md の紹介サイトを実際に制作しました。

**公開URL**: https://vibe-design-five.vercel.app

### サイト構成

| ページ | 内容 |
|--------|------|
| **Top** | DESIGN.mdのコンセプト紹介。AIエージェント時代のデザイン再現手法として位置づけ |
| **Features** | カラー・タイポグラフィ・レイアウト・スペーシング・コンポーネントの5軸を詳細解説 |
| **Principles** | デザインの4大原則（近接・整列・反復・対比）+ 5段階のビジュアルヒエラルキー |
| **Gallery** | 60種類のDESIGN.mdテンプレートをLive Previewで閲覧・ダウンロード可能 |
| **Get Started** | 3ステップの導入ガイド + FAQ |

### 技術スタック

- **フロントエンド**: HTML / CSS / Vanilla JS（フレームワークなし）
- **ホスティング**: Vercel
- **リポジトリ**: GitHub (south0120/VibeDesign)
- **AI支援**: Claude Code（DESIGN.md に基づくコード生成）

### ギャラリーの特徴

60種類のデザインテンプレートを収録。世界のWebサイト参考20種 + デザインスタイル40種。

**収録スタイル例**:
Stripe / Linear / Notion / Vercel / Spotify / Airbnb / リキッドグラス / ニューモーフィズム / ブルータリズム / ヴェイパーウェイブ / サイバーパンク / グラスモーフィズム / ホログラフィック / ダークネオン東京 / ペーパーカットアウト / レトロゲーム / 禅 など

各テンプレートには以下の機能を実装:
- **Live Preview**: design.mdの値を解析してリアルタイムでサンプルページを描画
- **WCAG 2.2スコア**: W3C基準のコントラスト比を自動計算・表示
- **ダウンロード**: .mdファイルを直接ダウンロードしてプロジェクトで即使用可能
- **CSSコンポーネント例**: リキッドグラス、ニューモーフィズム等はコピペ可能なCSS付き

## まとめ

DESIGN.md は「デザインの民主化」を実現するツールだと感じました。

- デザイナーがいなくても、テンプレートを選ぶだけで一貫性のあるUIが作れる
- エンジニアが自然言語でAIに指示するだけで、デザインシステムに準拠したコードが生成される
- Markdownという最もシンプルなフォーマットで、最も複雑なデザイン課題を解決できる

**AIエージェントの時代に、デザインの「仕様書」は .fig でも .sketch でもなく、.md になる。** そう確信したセミナーでした。
